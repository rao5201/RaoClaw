## 🚀 快速部署

```bash
# 1. 克隆项目
git clone https://github.com/rao5201/RaoClaw.git
cd RaoClaw

# 2. 配置环境
cp .env.example .env

# 3. 启动服务
docker-compose up -d

# 4. 访问网站
# http://localhost:8080
```
